# Artist Portfolio Website

This project is a custom portfolio website developed for a visual artist. The application was built as a full-stack solution using React on the frontend and Sanity CMS on the backend.

The website is designed to showcase the artist’s work, biography, and contact information through a modern, responsive interface.

---

## Project Overview

The goal of the project was to create a clean and visually focused portfolio platform that allows the artist to manage content independently through a headless CMS.

The frontend was fully styled from scratch using custom CSS, with emphasis on responsive design and smooth user experience.

---

## Features

* responsive artist portfolio layout
* dynamic content fetched from Sanity CMS
* custom CSS styling (no UI frameworks)
* image gallery for artworks
* artist biography section
* contact section
* reusable React components
* clean and minimal UI

---

## Technology Stack

* Frontend: React
* Styling: Custom CSS (from scratch)
* CMS: Sanity (headless CMS)
* Deployment: Self-hosted
